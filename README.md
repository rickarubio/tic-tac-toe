# Tic Tac Toe
## Running locally
Install foreman if you haven't already:  
``` gem install foreman ```  
Then:  
``` bundle install ```  
Finally:  
``` foreman s ```  
  
The application will be available at [http://localhost:5000](http://localhost:5000)
